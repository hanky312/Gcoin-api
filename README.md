# Gcoin-api
## Requirement
    Python 2.7
    virtualenv # pip install virtualenv
## Run server
### Activate virtualenv
    $ source ./env/bin/activate
    $ pip install -r requirements.txt
### Start server
    $ python bottle-test.py
### Stop server
    $ CTRL+C
    $ deactivate
