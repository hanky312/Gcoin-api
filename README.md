# Gcoin-api
## Requirement
    Python 2.7
    virtualenv # pip install virtualenv
## Run server
### Activate virtualenv
    $ virtualenv .env
    $ source ./env/bin/activate
    $ pip install -r requirements.txt
### Start server
    $ python bottle-test.py
### Stop server
    $ Hit Ctrl-C to quit
    $ deactivate
